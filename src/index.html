Other Useful Vim Commands

  • :w — Save file
  • :q — Quit Vim
  • :wq — Save and quit
  • /foo — Find foo in file
  • :%s/foo/bar/g — Replace foo with bar in whole file
  • :s/foo/bar/g — Replace foo with bar in current line
  • :help — Open Vim help
  • 0 — Go to start of line
  • $ — Go to end of line
